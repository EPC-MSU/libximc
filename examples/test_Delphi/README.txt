How to work with example:
1. The library with dependencies is located in the ximc/winX folders. The following files are required for the example to work: libximc.dll, xibridge.dll. They need to be copied to the folder with the example
2. Run test_Delphi.dpr using your Delphi IDE and compile the example
3. To run the compiled example from the command line, run test_Delphi.exe
